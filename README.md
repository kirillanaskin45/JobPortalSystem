## 📋 О проекте

**Job Portal Management System** — это полнофункциональная система для управления заявками на вакансии. Проект реализован в двух версиях: десктопное приложение на Java Swing и веб-приложение на HTML/CSS/JavaScript.

### 🎯 Цель проекта
Создать удобный инструмент для HR-специалистов и соискателей для управления процессом подачи и обработки заявок на вакансии.

## ✨ Возможности

### 📝 Управление заявками
- ✅ **Добавление заявок** — подача новой заявки с валидацией данных
- ✅ **Просмотр всех заявок** — табличное отображение с сортировкой
- ✅ **Редактирование** — изменение существующих заявок
- ✅ **Удаление** — удаление заявок с подтверждением
- ✅ **Поиск** — фильтрация по имени, должности или email
- ✅ **Изменение статуса** — управление статусом заявки

### 📊 Статусы заявок
- 🆕 **Новая** — только поступившая заявка
- 🔍 **Рассматривается** — в процессе рассмотрения
- 📞 **Интервью** — назначено собеседование
- ✅ **Принята** — успешное прохождение
- ❌ **Отклонена** — отказ

### 🎨 Дополнительные функции
- 📊 **Статистика** — подсчет заявок по статусам
- 🔔 **Уведомления** — информационные сообщения о действиях
- 📱 **Адаптивность** — корректное отображение на всех устройствах
- 🎨 **Цветовое кодирование** — визуальное выделение статусов

## 🖥 Версии приложения

### 1. Десктоп версия (Java Swing)
- **Технологии**: Java, Swing, AWT
- **Интерфейс**: Нативное окно с разделенной панелью
- **Особенности**:
  - Градиентный заголовок
  - Цветовое выделение статусов
  - Валидация email и телефона
  - JOptionPane для уведомлений

### 2. Веб версия (HTML/CSS/JS)
- **Технологии**: HTML5, CSS3, JavaScript ES6
- **Интерфейс**: Адаптивный дизайн
- **Особенности**:
  - Модальное окно для изменения статуса
  - Анимации при появлении записей
  - Гибкая верстка для мобильных устройств
  - Визуальные эффекты при наведении

## 📊 Структура данных

Каждая заявка содержит:
- **ФИО** — полное имя соискателя
- **Email** — электронная почта
- **Телефон** — контактный номер
- **Должность** — желаемая позиция
- **Квалификация** — образование/навыки
- **Статус** — текущее состояние заявки
- **Дата** — дата подачи
